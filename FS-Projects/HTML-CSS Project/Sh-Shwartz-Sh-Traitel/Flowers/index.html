אצלנו תוכלו למצוא את כל סוגי הצמחים, החל בעציצים וכלה בצמחיה לגינה או לפנטהאוז: צמחי בית או גינה, צמחי חורף או קיץ, פרחים טריים, צמחיה מלאכותית, פרחי בר ועוד. זאת לצד מגוון אביזרים משלימים, הכוללים אדניות, עציצים, כלי חרס וקרמיקה, אגרטלים, זאת לצד מגוון אביזרים משלימים מתנות. במשתלה שלנו פועלת מחלקה מקצועית לזרים ולסידורי פרחים, המבצעת זרי כלה, קישוטי רכב, עיצובי שולחנות לאירועים, מחי חורף או קיץ זרי פרחים בהתאמה אישית ועוד.
